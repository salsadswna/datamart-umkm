# DataMart: Prediksi Transaksi UMKM di Marketplace Indonesia

Proyek datathon AI Impact Challenge - Microsoft Elevate Training Center x Dicoding

## Dataset
Dataset yang digunakan: [Indonesia E-Commerce Sales & Shipping 2023-2025](https://www.kaggle.com/datasets/bakitacos/indonesia-e-commerce-sales-shipping-2023-2025) (Kaggle)

## Cara Menjalankan
1. Download dataset dari link Kaggle di atas
2. Buka notebook DataMart_EDA.ipynb via tombol "Open in Colab"
3. Upload file `all_months_clean.csv` ke Colab session
4. Run all cells

## Tech Stack
- Python, scikit-learn, pandas, matplotlib, seaborn
- Microsoft Azure Machine Learning
- Dataset: Kaggle (publik)
